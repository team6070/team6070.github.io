Litegapps is a custom gapps (open source project) designed to be as light and minimal as possible, so that it uses less RAM memory and battery consumption.

LiteGapps has a system (systemless) or (non systemless), by supporting :

Architecture : arm,arm64

Android version : 5.0,5.1,6.0,7.0,7.1,8.0,8.1,9.0,10.0,11.0,12.0 or later

in one package,and even in one package there is an uninstaller (without clean wipe). LiteGapps has 3 versions flashable namely :

[MAGISK] = Magisk Module Flashable

[RECOVERY] = Recovery Module Flashable

[AUTO] = Magisk/Recovery Module Flashable

The beginnings of Litegapps

Litegapps is inspired by my habit of always being simple, for years I was looking for the lightest Gapps and I found them but it's not as light as I thought. actually I don't like GoogleApps because it only takes up RAM, battery and makes your cellphone slow. I only need gapps to sync contacts and download games after that I don't need gapps anymore, so I made this Litegapps! With Litegapps you can uninstall all Gapps if you don't need them from your phone and your life! Because this is systemless.

Unique installer

Litegapps uses a unique installer of which you can :

• Install as magisk module/recovery flashable in one package

• Can install/uninstall in magisk manager/recovery in one package

• Supports many versions of android and architecture in one package with a small fixed size

• can uninstall cleanly in one package (without additional uninstaller)

LiteGapps

LiteGapps is specifically targeted at architecture and this particular android version has a small size of around 47MB+.but there is also a ALL version which supports all android versions in one package but has a large size.

Architecture : arm,arm64

Android version : 5.0,5.1,6.0,7.0,7.1,8.0,8.1,9.0,10.0,11.0,12.0

LiteGapps Pixel

LiteGapps Pixel is specifically targeted at architecture and this particular android version has a small size of around 47MB+.but there is also a ALL version which supports all android versions in one package but has a large size.

This is a derivative of "litegapps" which distinguishes google apps more fully ported from the Google Pixel ROM.

This is intended for those who are looking for a more complete Google Apps.

Architecture : arm64

Android version : 11.0,12.0

LiteGapps++ (double plus)

Litegapps++ (double plus) is the development of "litegapps" which allows it to support almost all versions of android and architecture while maintaining its small size.

Architecture = arm,arm64

Android = 8.1,9.0,10.0,11.0,12.0

LiteGapps++ LTS

Litegapps++ LTS is the development of "litegapps" which allows it to support almost all versions of android and architecture while maintaining its small size.

It's a derivative of "litegapps++" distinguishing it's made for the long term

This is intended for those who rarely update "litegapps" so they can use it for a long time.

Architecture = arm,arm64

Android = 8.1,9.0,10.0,11.0,12.0

LiteGapps++ MicroG

Litegapps++ MicroG is the development of "litegapps" which allows it to support almost all versions of android and architecture while maintaining its small size.

It is a derivative of "litegapps++" which distinguishes it from using MicroG GMS and several other properties, thus consuming less RAM and battery size and usage

This is intended for those who are looking for a lighter version of Litegapps using battery and RAM.

Architecture = arm,arm64

Android = 8.1,9.0,10.0,11.0,12.0

LiteGapps Controller
