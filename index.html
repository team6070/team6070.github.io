Litegapps is a custom gapps with (open source and free) for smartphones based on Google Pixel Devices that has support across a wide range of devices.

small friendly complete

LiteGapps first release on 12 july 2020 supports android Lollipop Up to Android 12. comes with 3 products, namely LiteGapps,LiteGapps++,LiteGapps Controller.

LiteGapps has systemless and non systemless installation system. using 3 installation modes :

[MAGISK] = Magisk Module Flashable

[RECOVERY] = Recovery Module Flashable

[AUTO] = Magisk/Recovery Module Flashable

LiteGapps

LiteGapps is specifically targeted for certain android/architecture versions, which are based on Google Pixel devices. It has 6 variants, namely Lite,Bakso,Core,Micro,Pixel.

Support :

Architecture = arm,arm64

Android = 5.0 to 12.0

LiteGapps++

LiteGapps++ litegapps is a development of regular litegapps, with litegapps++ you can use it for cross-architectures and android versions in 1 flashable ZIP, while maintaining its small size. It has 3 variants, namely Reguler,LTS,MicroG.

Support :

Architecture = arm,arm64

Android = 5.0 to 12.0

LiteGapps Controller

LiteGapps Controller is an additional feature of litegapps or litegapps ++ which aims to fully control litegapps, which contains additional features such as.

• Booster

• Gaming Mode

• Download Package Addon

• battery saver

• Tweaks

• And many others
